Detailed Description

This is WorkerThread of application, it does hard work in different thread as gui runs in This class is not crossplatform! For crossplatform solution replace ShellExecuteEx and WaitForSingleObject calls in transcodeFile() and related variables. Also add folder separators / everywhere where is \ used.
